# ultramemory-mcp

**UltraMemory — one memory for every AI you use: Claude, ChatGPT, Perplexity, Gemini, Cursor, OpenClaw, Hermes, and any MCP client.** Tell it something once and it stays told — and when it doesn't know, it says so instead of making things up.

This is the documented install name for the UltraMemory client. The implementation ships in [`ultramemory-hermes`](https://pypi.org/project/ultramemory-hermes/), which this package installs automatically — you get the same `ultramemory` command and Python module either way.

```bash
pip install ultramemory-mcp
ultramemory configure
```
